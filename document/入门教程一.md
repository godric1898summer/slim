### 入门教程（一）安装slim

* 创建项目文件夹
```bash
/User/akio/develop/slim

```

* 创建composer.json
```json
{
    "require": {
        "slim/slim": "3.0"
    }
}
```

* 安装
```bash
composer install
```
或
```bash
composer require slim/slim /User/akio/develop/slim
```

* 基础目录
```text
# 我的目录结构
slim|
    bin     # 可执行文件位置
    config  # 配置文件位置
    public  # 入口文件
    src     # 应用
    temp    # 模版文件，API应用不需要该目录
    var     # 日志、缓存等
```
